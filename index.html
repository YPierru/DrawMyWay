Présentation

DrawMyWay est une application androïd, destinée aussi bien aux coureurs à pied qu'aux automobilistes. DrawMyWay vous permet de tracer vous-même vos trajets sur une carte, l'application vous guidera à la manière d'un GPS. Vous pouvez choisir de créer un trajet entre 2 points, ou entre une série de points. L'accent est mis sur la simplicité d'utilisation et l'efficacité.

Technologies utilisées... et pourquoi

  • Androïd pour sa facilité d'approche étant donné que nous connaissions déjà le Java.
  • Google Map, bien qu'il existe d'autres systèmes de cartographie (IGN, OpenStreetMap...) Google Map était le seul combinant plusieurs avantages : gratuité, précision, grosse communauté de développeurs, beaucoup de doc/tutoriels, beaucoup de retours.
  • Google Directions API, liée à Google Map, permet de récupérer des instructions de déplacements entre 2 points donnés.
  • ActionBarSherlock, librairie utilisée pour tous les éléments graphiques. C'est une solution temporaire car les développements évolutifs ont été stoppés. A terme une migration est prévue vers ActionBarCompat, l'API Google.
  • Commons-io, librairie créée par Apache, permettant de gérer les entrées/sorties.
  • Gson, librairie Google, permettant de parser un fichier JSON vers une structure Java.
  • Joda-time, librairie permettant de manipuler, plus facilement qu'en java natif, les dates et heures.
  • JSoup, librairie permettant de parser du contenu HTML. Elle est utilisée pour supprimer les balises HTML d'une chaine de caractères.
  • Simple-side-drawer, librairie permettant d'afficher des panneaux latéraux dans nos écrans. C'est la librairie la plus simple à utiliser que nous ayons trouvé.

Avancement

Fait

  • Avoir une carte (GoogleMap) à l'écran.
  • Modifier à sa guise le type de la carte (normal, terrain, satellite, hybride).
  • Pouvoir dessiner un trajet (départ/arrivé, jalons intermédiaire et validation).
  • Sauvegarder un trajet.
  • Avoir un GPS purement textuel, reprenant les informations de directions du trajet.
  • Afficher la liste des trajets sauvegardés.
  • Supprimer un trajet sauvegardé.
  • Modifier le nom d'un trajet sauvegardé.
  • Afficher les détails d'un trajet (kilométrage, temps prévu, date de création/modification etc).
  • Permettre de modifier un trajet en déplaçant des jalons, les supprimant etc (correction mode).

En cours

  • Permettre de rechercher un lieu avec une auto-complétion.
  • Modifications ergonomiques par rapport à l'interface utilisateur.

A Faire

  • Avoir un GPS graphique, avec prise en compte de la position de l'utilisateur et de ses déplacements
  • Avoir un GPS vocal à partir des données recueillies à la validation d'un trajet
  • Mieux gérer la validation d'un trajet (Thread)
  • Tester sur différents appareils (taille d'écran et version d'Androïd)
  • Dessin d'icônes, embellissement graphique

Arborescence du repo

APK

Ce dossier contient le .apk de l'application. Pour l'installer sur votre terminal android :

  • Etre en version 4.0 ou ultérieur
  • Connectez votre terminal à votre ordinateur
  • Déposer le .apk dans votre terminal (à la racine par exemple)
  • Sur votre terminal, à l'aide d'un file manager, naviguez jusqu'au .apk
  • Tapez dessus et confirmez que vous voulez bien installer l'application
